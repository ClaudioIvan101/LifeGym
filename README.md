# LifeGym- Sistema para Gestión de Gimnasios
Grupo: 8

Integrantes: 
- Claudio Ivan Gomez (rama-claudio)
- Lautaro Oporto (rama-lautaro)
- Gustavo Vernengo (rama-gustavo)

Este proyecto es un sistema de gestió para un gimnasio, desarrollado en Java 
Permite gestionar socios, membresías y asistencias, vencimientos integrando los conceptos de POO, 
colecciones, excepciones y programación estructurada.
El trabajo fue realizado de forma colaborativa utilizando Git, 
donde cada integrante aportó funcionalidades desde su propia rama.
Este proyecto fue pensado con el objetivo de resolver un problema real, como lo es la cuestion de asistencias para controlar
vencimientos de las actividades que ofrece un gimnasio a sus socios.
A su vez que sea un proyecto completamente escalable a futuro y que permita agregar nuevas funcionalidades.
Se pueden registrar socios, administrar membresías, marcar asistencias, control de vencimiento de cuota
y realizar varias consultas.
